## Get space details

To grab info about a space you can use the _getSpace_ GraphQL query. Be sure to switch "space-id" to the actual id youâ€™re curious about.

```
query getSpace {
  space(id: "space-id") {
    title
    description
    membersUpdated
    members {
      items {
        email
        displayName
      }
    }
    conversation{
      messages{
        items {
          content
        }
      }
    }
  }
}
```
